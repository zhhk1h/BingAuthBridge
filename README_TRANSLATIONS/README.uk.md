# Bing Auth Bridge (Міст автентифікації Bing) v1.0

Це спеціалізоване розширення для браузера з відкритим кодом для експорту та імпорту вашого статусу входу в Microsoft Bing та Обліковий запис Microsoft одним клацанням миші.

## Навіщо цей інструмент?

При зміні браузера (наприклад, з Chrome на Edge) або налаштуванні нового комп'ютера, перенесення вашого статусу входу — особливо балів Bing Rewards — є виснажливим процесом. Цей інструмент автоматизує це перенесення шляхом безпечної передачі відповідних файлів cookie.

## Чому він на 100% безпечний та надійний?

Цей плагін був розроблений з урахуванням вашої конфіденційності та безпеки як найвищого пріоритету. Ми гарантуємо це завдяки двом стовпам: "Наші зобов'язання" та "Технічна прозорість".

### 1. Наші ключові зобов'язання щодо безпеки

Ми даємо вам ці обіцянки:

* ✅ **Ніколи не аналізується**: Цей плагін ніколи не читає, не перевіряє та не аналізує **вміст** ваших даних cookie. Він розглядає ваші дані як "чорну скриньку", яку потрібно перемістити, а не відкрити.
* ✅ **Ніколи не змінюється**: Цей плагін не *модифікує* та не *змінює* ваші дані під час процесу експорту/імпорту. Експортований файл є точною копією даних вашого браузера, і він записується назад точно таким, яким був.
* ✅ **Ніколи не передається**: Цей плагін **не містить мережевого коду**. Всі операції (експорт та імпорт) відбуваються на 100% на вашій локальній машині. Ваші дані **ніколи** не надсилаються на жоден зовнішній сервер.

### 2. Наша технічна прозорість (можна перевірити)

Ми забезпечуємо виконання вищевказаних обіцянок завдяки нашому коду та дизайну дозволів:

* **Мінімальні дозволи**:
    Цей плагін **лише** запитує абсолютний мінімум дозволів, необхідних для функціонування. Ви можете перевірити це у файлі `manifest.json`:
    ```json
      "host_permissions": [
        "*://*[.bing.com/](https://.bing.com/)",
        "*://*[.microsoft.com/](https://.microsoft.com/)",
        "*://*[.live.com/](https://.live.com/)"
      ]
    ```
    Це означає, що плагін технічно **нездатний** отримати доступ або змінити ваші дані на **будь-якому іншому сайті** (наприклад, у вашому банку, електронній пошті чи соціальних мережах).

* **Відкритий код, що можна перевірити**:
    Ви можете перевірити кожний рядок коду в `popup.js`. Ви не знайдете жодних мережевих запитів `fetch()` або `XMLHttpRequest` та жодної логіки для аналізу даних.

## Як встановити (Sideloading)

Оскільки цей плагін є відкритим, ви можете завантажити його безпосередньо з вихідного коду.

1.  **Завантажити:**
    * Натисніть зелену кнопку `Code` у верхній частині цієї сторінки та виберіть `Download ZIP`.
2.  **Розпакувати:**
    * Розпакуйте `.zip` файл у постійну папку, яку ви **не будете видаляти** (наприклад, `D:\MyPlugins\BingAuthBridge`).
3.  **Завантажити в Chrome / Edge:**
    * Відкрийте ваш браузер (Chrome або Edge).
    * Введіть `chrome://extensions` (для Chrome) або `edge://extensions` (для Edge) в адресний рядок і натисніть Enter.
    * Увімкніть перемикач **"Режим розробника"** у верхньому правому (Chrome) або нижньому лівому (Edge) куті.
    * Натисніть кнопку **"Завантажити розпаковане"** (Load unpacked).
    * У вікні вибору файлів виберіть папку, яку ви щойно розпакували (наприклад, `D:\MyPlugins\BingAuthBridge`).
4.  **Готово:** Іконка плагіна з'явиться на вашій панелі інструментів.

## Як використовувати

1.  **У вашому вихідному браузері (напр. Chrome):**
    * Переконайтеся, що ви ввійшли в Bing.
    * Натисніть на іконку плагіна на панелі інструментів.
    * Натисніть кнопку "1. Експортувати Cookies". Файл з назвою `bing_cookies.json` буде збережено.
2.  **У вашому цільовому браузері (напр. Edge):**
    * (Поки що не входьте в Bing).
    * Натисніть на іконку плагіна на панелі інструментів.
    * Натисніть кнопку "2. Імпортувати Cookies" і виберіть файл `bing_cookies.json`, який ви щойно зберегли.
    * Після повідомлення про успіх, оновіть сторінку `bing.com`. Тепер ви будете в системі.

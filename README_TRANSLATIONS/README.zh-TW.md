# Bing Auth Bridge (必應登入遷移助手) v1.0

這是一款專門的、開源的瀏覽器擴充功能，用於一鍵匯出和匯入您的 Microsoft Bing 和 Microsoft 帳戶的登入狀態。

## 為什麼需要此工具？

當您更換瀏覽器（例如從 Chrome 遷移到 Edge）或在新電腦上設定瀏覽器時，手動遷移您的登入狀態（尤其是 Bing 獎勵積分）是一個繁瑣的過程。此工具通過安全地傳輸相關的 Cookies 來自動化該遷移過程。

## 為什麼它 100% 安全可信？

本擴充功能在設計時將您的隱私和安全置於最高優先級。我們透過「我們的承諾」和「技術透明度」兩大支柱來保證：

### 1. 我們的核心安全承諾

我們向您鄭重承諾：

* ✅ **絕不分析**：本擴充功能絕不會讀取、檢查或分析您 Cookie 資料中的**任何內容**。它將您的資料視為一個無法打開的「黑盒子」，只負責「搬運」，而非「開啟」。
* ✅ **絕不篡改**：本擴充功能不會在匯出或匯入過程中 *修改* 或 *篡改* 您的資料內容。匯出的檔案是您瀏覽器資料的精確副本，匯入時也會原樣寫入。
* ✅ **絕不傳輸**：本擴充功能**不包含任何網路程式碼**。所有操作（匯出和匯入）均在您的本機電腦上 100% 完成。您的資料**永遠不會**被發送到任何外部伺服器。

### 2. 我們的技術透明度（可驗證）

我們透過程式碼和權限設計，在技術上確保了上述承諾的實現：

* **最小權限原則**：
    本擴充功能**僅**申請了其運作所必需的絕對最小權限。您可以在 `manifest.json` 檔案中驗證：
    ```json
      "host_permissions": [
        "*://*[.bing.com/](https://.bing.com/)",
        "*://*[.microsoft.com/](https://.microsoft.com/)",
        "*://*[.live.com/](https://.live.com/)"
      ]
    ```
    這意味著本擴充功能在技術上**無法**存取或修改您在**任何其他網站**（如您的銀行、電子郵件或社交媒體）上的資料。

* **可驗證的開源程式碼**：
    您可以審核 `popup.js` 中的每一行程式碼。您會發現零 `fetch()` 或 `XMLHttpRequest` 網路請求，也零資料分析的邏輯。

## 如何安裝（側載）

由於本擴充功能是開源的，您可以直接從原始碼載入它。

1.  **下載：**
    * 點選此頁面頂部的綠色 `Code` 按鈕，然後選擇 `Download ZIP`。
2.  **解壓縮：**
    * 將 `.zip` 檔案解壓縮到一個您**不會刪除**的永久資料夾（例如：`D:\MyPlugins\BingAuthBridge`）。
3.  **在 Chrome / Edge 中載入：**
    * 開啟您的瀏覽器（Chrome 或 Edge）。
    * 在網址列輸入 `chrome://extensions` (Chrome) 或 `edge://extensions` (Edge) 並按 Enter。
    * 啟用位於右上角 (Chrome) 或左下角 (Edge) 的 **「開發人員模式」** 開關。
    * 點選 **「載入未封裝的擴充功能」** 按鈕。
    * 在檔案提示框中，選擇您剛剛解壓縮的那個資料夾（例如：`D:\MyPlugins\BingAuthBridge`）。
4.  **完成：** 擴充功能圖示將出現在您的工具列中。

## 如何使用

1.  **在您的來源瀏覽器（例如 Chrome）中：**
    * 確保您已登入 Bing。
    * 點選工具列中的擴充功能圖示。
    * 點選「1. 匯出 Cookies」按鈕。一個名為 `bing_cookies.json` 的檔案將被儲存。
2.  **在您的目標瀏覽器（例如 Edge）中：**
    * （暫時不要登入 Bing）。
    * 點選工具列中的擴充功能圖示。
    * 點選「2. 匯入 Cookies」按鈕，並選擇您剛剛儲存的 `bing_cookies.json` 檔案。
    * 報告成功後，重新整理 `bing.com` 頁面。您會發現您現在已經登入了。

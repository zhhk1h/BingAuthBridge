# Bing Auth Bridge (Bing 인증 브리지) v1.0

이것은 Microsoft Bing 및 Microsoft 계정 로그인 상태를 원클릭으로 내보내고 가져오기 위한 전용 오픈소스 브라우저 확장 프로그램입니다.

## 이 도구가 필요한 이유

브라우저를 변경하거나(예: Chrome에서 Edge로) 새 컴퓨터를 설정할 때 로그인 상태, 특히 Bing Rewards 포인트를 마이그레이션하는 것은 지루한 과정입니다. 이 도구는 관련 쿠키를 안전하게 전송하여 해당 마이그레이션을 자동화합니다.

## 100% 안전하고 신뢰할 수 있는 이유

이 플러그인은 사용자의 개인 정보 보호와 보안을 최우선으로 설계되었습니다. 저희는 "우리의 약속"과 "기술적 투명성"이라는 두 가지 기둥을 통해 이를 보장합니다.

### 1. 핵심 보안 약속

저희는 사용자에게 다음과 같이 약속합니다:

* ✅ **절대 분석하지 않음**: 이 플러그인은 쿠키 데이터의 **내용**을 절대 읽거나, 검사하거나, 분석하지 않습니다. 데이터를 '열 수 없는 블랙박스'로 취급하며, 열지 않고 이동만 시킵니다.
* ✅ **절대 변경하지 않음**: 이 플러그인은 내보내기/가져오기 과정에서 데이터를 *수정*하거나 *변경*하지 않습니다. 내보낸 파일은 브라우저 데이터의 정확한 사본이며, 있는 그대로 다시 작성됩니다.
* ✅ **절대 전송하지 않음**: 이 플러그인은 **어떠한 네트워크 코드도 포함하고 있지 않습니다**. 모든 작업(내보내기 및 가져오기)은 100% 사용자의 로컬 컴퓨터에서 이루어집니다. 사용자의 데이터는 **절대로** 외부 서버로 전송되지 않습니다.

### 2. 기술적 투명성 (검증 가능)

저희는 위의 약속을 코드 및 권한 설계를 통해 기술적으로 강제합니다:

* **최소한의 권한**:
    이 플러그인은 작동에 필요한 **절대 최소한의 권한**만 요청합니다. 이는 `manifest.json` 파일에서 확인할 수 있습니다:
    ```json
      "host_permissions": [
        "*://*[.bing.com/](https://.bing.com/)",
        "*://*[.microsoft.com/](https://.microsoft.com/)",
        "*://*[.live.com/](https://.live.com/)"
      ]
    ```
    이는 이 플러그인이 기술적으로 **다른 어떤 사이트** (은행, 이메일, 소셜 미디어 등)의 데이터에도 접근하거나 수정할 수 **없음**을 의미합니다.

* **검증 가능한 오픈소스 코드**:
    `popup.js`의 모든 코드 라인을 감사할 수 있습니다. `fetch()` 또는 `XMLHttpRequest` 네트워크 요청이나 데이터 분석 로직을 전혀 찾을 수 없을 것입니다.

## 설치 방법 (사이드로딩)

이 플러그인은 오픈소스이므로 소스 코드에서 직접 로드할 수 있습니다.

1.  **다운로드:**
    * 이 페이지 상단의 녹색 `Code` 버튼을 클릭하고 `Download ZIP`을 선택합니다.
2.  **압축 해제:**
    * `.zip` 파일을 **삭제하지 않을** 영구 폴더(예: `D:\MyPlugins\BingAuthBridge`)에 압축을 풉니다.
3.  **Chrome / Edge에서 로드:**
    * 브라우저(Chrome 또는 Edge)를 엽니다.
    * 주소창에 `chrome://extensions` (Chrome) 또는 `edge://extensions` (Edge)를 입력하고 Enter 키를 누릅니다.
    * 오른쪽 상단(Chrome) 또는 왼쪽 하단(Edge)의 **"개발자 모드"** 토글을 활성화합니다.
    * **"압축 해제된 확장 프로그램을 로드합니다"** (Load unpacked) 버튼을 클릭합니다.
    * 파일 프롬프트에서 방금 압축을 푼 폴더(예: `D:\MyPlugins\BingAuthBridge`)를 선택합니다.
4.  **완료:** 플러그인 아이콘이 툴바에 나타납니다.

## 사용 방법

1.  **소스 브라우저 (예: Chrome)에서:**
    * Bing에 로그인되어 있는지 확인합니다.
    * 툴바의 플러그인 아이콘을 클릭합니다.
    * "1. 쿠키 내보내기" 버튼을 클릭합니다. `bing_cookies.json`이라는 이름의 파일이 저장됩니다.
2.  **대상 브라우저 (예: Edge)에서:**
    * (아직 Bing에 로그인하지 마십시오).
    * 툴바의 플러그인 아이콘을 클릭합니다.
    * "2. 쿠키 가져오기" 버튼을 클릭하고 방금 저장한 `bing_cookies.json` 파일을 선택합니다.
    * 성공 메시지가 나타나면 `bing.com` 페이지를 새로고침합니다. 이제 로그인된 상태가 됩니다.

# Bing Auth Bridge (必应登录迁移助手) v1.0

这是一个专用的、开源的浏览器插件，用于一键导出和导入微软必应 (Bing) 和微软账户 (Microsoft Account) 的登录状态。

## 为什么需要这个工具？

当您更换浏览器（例如从 Chrome 迁移到 Edge）或在新电脑上设置浏览器时，手动迁移登录状态（尤其是必应积分）非常繁琐。此工具通过导出/导入相关的 Cookies，实现了“一键迁移”。

## 为什么它是 100% 安全可信的？

本插件在设计时将您的隐私和安全置于首位。我们通过“技术限制”和“道德承诺”两方面来保证：

### 1. 我们的核心安全承诺

我们向您郑重承诺：

* ✅ **绝不分析**：本插件不会读取、检查或分析您 Cookie 数据中的**任何内容**。它将您的数据视为一个无法打开的“黑盒”，只做“搬运”。
* ✅ **绝不篡改**：本插件不会在导出或导入过程中，*修改*或*篡改*您的数据内容。导出的文件是您浏览器数据的精确副本，导入时也会被原样写入。
* ✅ **绝不传输**：本插件**不包含任何网络请求代码**。所有操作（导出和导入）均在您的本地浏览器上完成。您的数据**永远不会**被发送到任何外部服务器。

### 2. 我们的技术透明度（可验证）

我们通过代码和权限设计，在技术上确保了上述承诺的实现：

* **最小权限原则**：
    本插件**只**申请了它运行所必需的最小权限。您可以在 `manifest.json` 文件中看到：
    ```json
      "host_permissions": [
        "*://*[.bing.com/](https://.bing.com/)",
        "*://*[.microsoft.com/](https://.microsoft.com/)",
        "*://*[.live.com/](https://.live.com/)"
      ]
    ```
    这意味着本插件在技术上**无法**访问或修改您在**任何其他网站**（如您的银行、邮箱等）上的数据。

* **代码开源可查**：
    您可以检查 `popup.js` 中的代码，确认没有任何 `fetch()` 或 `XMLHttpRequest` 网络请求，也没有任何分析或修改数据内容的逻辑。

## 如何安装 (通过开发者模式)

本插件是开源的，您可以直接从源代码加载，完全绕开应用商店。

1.  **下载代码：**
    * 点击本页面右上角的 `Code` 按钮，选择 `Download ZIP`。
2.  **解压缩：**
    * 将下载的 `.zip` 解压缩到一个**您不会删除**的永久文件夹（例如：`D:\MyPlugins\BingAuthBridge`）。
3.  **在 Chrome / Edge 中加载：**
    * 打开您的浏览器（Chrome 或 Edge）。
    * 在地址栏输入：`chrome://extensions` (Chrome) 或 `edge://extensions` (Edge) 并回车。
    * 在页面**右上角 (Chrome)** 或 **左下角 (Edge)**，打开 **“开发者模式”** 的开关。
    * 点击 **“加载已解压缩的扩展程序”** (Load unpacked) 按钮。
    * 在弹出的窗口中，选择您刚才解压的那个文件夹（`D:\MyPlugins\BingAuthBridge`）。
4.  **完成：** 插件图标会出现在您的工具栏上。

## 如何使用

1.  **在源浏览器 (例如 Chrome) 中：**
    * 确保您已登录必应。
    * 点击工具栏上的插件图标。
    * 点击 “1. 导出 Cookies” 按钮。一个名为 `bing_cookies.json` 的文件将被下载。
2.  **在目标浏览器 (例如 Edge) 中：**
    * （先不要登录必应）。
    * 点击工具栏上的插件图标。
    * 点击 “2. 导入 Cookies” 按钮，选择刚才下载的 `bing_cookies.json` 文件。
    * 导入成功后，刷新 `bing.com` 页面，您会发现已经自动登录了。

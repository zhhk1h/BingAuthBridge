# Bing Auth Bridge (Bing認証ブリッジ) v1.0

これは、Microsoft BingおよびMicrosoftアカウントのログイン状態をワンクリックでエクスポートおよびインポートするための、専用のオープンソース・ブラウザ拡張機能です。

## なぜこのツールが必要か？

ブラウザを切り替える（例：ChromeからEdgeへ）または新しいコンピュータをセットアップする際、ログイン状態（特にBingリワードポイント）を移行するのは面倒な作業です。このツールは、関連するCookieを安全に転送することで、その移行を自動化します。

## なぜ100%安全で信頼できるのか？

このプラグインは、お客様のプライバシーとセキュリティを最優先に設計されています。私たちは「私たちの誓約」と「技術的透明性」の二つの柱を通じてこれを保証します。

### 1. 私たちの核となるセキュリティ誓約

私たちは以下のことをお約束します：

* ✅ **決して分析しません**：このプラグインは、お客様のクッキーデータの**内容**を読み取ったり、検査したり、分析したりすることは一切ありません。お客様のデータを「開けられないブラックボックス」として扱い、移動させるだけです。
* ✅ **決して改ざんしません**：このプラグインは、エクスポート/インポートのプロセス中にお客様のデータを*変更*したり*改ざん*したりしません。エクスポートされるファイルはブラウザデータの正確なコピーであり、そのまま書き戻されます。
* ✅ **決して送信しません**：このプラグインは**ネットワークコードを含んでいません**。すべての操作（エクスポートとインポート）は100%ローカルマシン上で行われます。お客様のデータが外部サーバーに送信されることは**決して**ありません。

### 2. 私たちの技術的透明性（検証可能）

私たちは上記の誓約を、コードと権限の設計を通じて技術的に強制します：

* **最小権限の原則**：
    このプラグインは、機能するために必要な**絶対最小限の権限のみ**を要求します。これは`manifest.json`ファイルで確認できます：
    ```json
      "host_permissions": [
        "*://*[.bing.com/](https://.bing.com/)",
        "*://*[.microsoft.com/](https://.microsoft.com/)",
        "*://*[.live.com/](https://.live.com/)"
      ]
    ```
    これは、このプラグインが技術的に**他のどのサイト**（銀行、メール、SNSなど）のデータにもアクセスしたり変更したりすることが**不可能**であることを意味します。

* **検証可能なオープンソースコード**：
    `popup.js`のすべてのコード行を監査できます。ネットワークリクエスト（`fetch()`や`XMLHttpRequest`）やデータ分析ロジックは一切見つからないはずです。

## インストール方法（サイドローディング）

このプラグインはオープンソースであるため、ソースコードから直接ロードできます。

1.  **ダウンロード：**
    * このページの上部にある緑色の`Code`ボタンをクリックし、`Download ZIP`を選択します。
2.  **解凍：**
    * `.zip`ファイルを、**削除しない**恒久的なフォルダ（例：`D:\MyPlugins\BingAuthBridge`）に解凍します。
3.  **Chrome / Edgeでロード：**
    * ブラウザ（ChromeまたはEdge）を開きます。
    * アドレスバーに `chrome://extensions` (Chrome) または `edge://extensions` (Edge) と入力し、Enterキーを押します。
    * 右上（Chrome）または左下（Edge）にある**「デベロッパーモード」**のトグルを有効にします。
    * **「パッケージ化されていない拡張機能を読み込む」**（Load unpacked）ボタンをクリックします。
    * ファイルダイアログで、先ほど解凍したフォルダ（例：`D:\MyPlugins\BingAuthBridge`）を選択します。
4.  **完了：** プラグインのアイコンがツールバーに表示されます。

## 使い方

1.  **移行元のブラウザ（例：Chrome）で：**
    * Bingにログインしていることを確認します。
    * ツールバーのプラグインアイコンをクリックします。
    * 「1. クッキーをエクスポート」ボタンをクリックします。`bing_cookies.json`という名前のファイルが保存されます。
2.  **移行先のブラウザ（例：Edge）で：**
    * （まだBingにログインしないでください）。
    * ツールバーのプラグインアイコンをクリックします。
    * 「2. クッキーをインポート」ボタンをクリックし、先ほど保存した`bing_cookies.json`ファイルを選択します。
    * 成功が報告されたら、`bing.com`のページを更新します。これでログイン状態になっているはずです。
